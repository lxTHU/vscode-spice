# Features
## Done
- Comment **toggle**
- Syntax Highlighting of 
    - Comments `*|;|&` 
    - Circuits Elements `R|...` `V|...` `X|...` 
    - Expressions `abs()|...` `""` 
    - SPICE Commands `.subckt|...` `.tran|...` 
- Snippets
    - Basic `.ac/.dc/.tran`
    - `.meas rise/fall/delay`
